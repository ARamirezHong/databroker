databroker
==========

A unified interface to the various data sources at NSLS-II.

Implemented:

- `metadatastore <https://github.com/NSLS-II/metadatastore>`_
- `filestore <https://github.com/NSLS-II/filestore>`_

Planned:

- channelarchiver
